# HTML Resume Website

This project is a single-page resume website created using **only HTML** as part of an assignment.
No CSS or JavaScript has been used. The focus of this project is on clean structure, proper HTML
semantics, and readability.

## Project Overview
The resume includes the following sections:
- Header / Name
- About Me
- Skills
- Experience
- Projects
- Education
- Contact Details

All sections are structured using appropriate HTML tags such as headings, paragraphs, lists, tables,
and horizontal rules.

## Setup Steps
1. Clone the repository or download the project files.
2. Open the `index.html` file in any web browser.

## Usage
- Open the resume page in a browser to view the complete resume.
- Scroll through the page to see different sections like skills, experience, projects, and education.
